# 🎁 Les Caisses

### Introduction

Afin d’ajouter une dimension aléatoire à l’aventure, plusieurs c**aisses** sont disponibles sur Blocaria.\
Certaines sont permanentes, tandis que d’autres sont disponibles pour une durée limitée.

Certains items sont disponibles seulement dans ces caisses comme les cosmétiques et plusieurs tags.

### Fonctionnement

Les Caisses sont accessibles au <kbd><mark style="color:yellow;">/spawn<mark style="color:yellow;"></kbd>.\
Pour les ouvrir, il vous suffira de cliquer sur la caisse avec la clé. Cela lance l'animation et vous donne la récompense.

### Obtention des clés

Les clés peuvent être obtenues de différentes manières :

* Achat dans la [boutique ](la-boutique.md)avec des **Gemmes** <mark style="color:yellow;">**`/`**</mark><mark style="color:yellow;">`boutique`</mark>
* [Vote](../bienvenue-sur-blocaria/comment-voter-pour-blocaria.md) pour le serveur via la commande <mark style="color:yellow;">**`/vote`**</mark>
* Récompenses d’évènements
* Progression dans certains métiers
