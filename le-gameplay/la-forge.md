# ⚒️ La Forge

### La Forge

La Forge est une fonctionnalité permettant la réparation des objets via une interface dédiée.\
Elle est accessible à l’aide de la commande : <kbd><mark style="color:yellow;">/forgeron<mark style="color:yellow;"></kbd>.

L’interface comporte deux onglets distincts :

**Réparation des outils de rareté**

**Réparation des outils évolutifs**

### Obtention des poudres

Le concassage de géodes permet d'en obtenir, avec une probabilité variable :

**Poudre de perlimpinpin** ![](../.gitbook/assets/perlimpinpin_powder.png)

**Poudre enchantée** ![](../.gitbook/assets/enchant_perlimpinpin_powder.png)

### Système de réparation

#### Outils de rareté

La réparation des outils de rareté nécessite l’utilisation de **poudre corrompue**.\
Chaque utilisation restaure **25 %** de durabilité de l’objet.

#### Outils évolutifs

Les outils évolutifs peuvent être réparés à l’aide de :

**Poudre de perlimpinpin** : restaure **15 %** de durabilité.

**Poudre enchantée** : restaure **100 %** de durabilité (réparation complète).

### Procédure

Pour effectuer une réparation :

Placer l’objet à réparer dans l’emplacement prévu à cet effet.

Ajouter la poudre compatible dans l’emplacement dédié.

Cliquer sur le bouton **Réparer** afin de valider l’action.

### Incompatibilité

Si une poudre incompatible est associée à un objet :

Un message d’erreur s’affiche.

Celui-ci précise que la poudre utilisée n’est pas adaptée à l’objet sélectionné.

### La fonderie

La fonderie est une fonctionnalité qui vous permet de fondre vos outils personnalisés en poudre corrompue. \
\
La fonderie est accessible de plusieurs façons : la commande <kbd><mark style="color:yellow;">/fondre<mark style="color:yellow;"></kbd> ou encore le <kbd><mark style="color:yellow;">/forgeron<mark style="color:yellow;"></kbd> (accessible aussi grâce au PNJ du forgeron).\
Les outils peuvent être fondus, quelle que soit leur durabilité, et vous pourrez en fondre jusqu'à 28 par opération.

<summary>Taux de poudre corrompue</summary>

| Rareté                                                                               | Poudre corrompue            |
| ------------------------------------------------------------------------------------ | --------------------------- |
| Rare (canne à pêche, épée, fourche, hache, pioche).                                  | 1 poudre corrompue / item   |
| Epique (canne à pêche, épée, fourche, hache, pioche)                                 | 4 poudres corrompue / item  |
| Légendaire (canne à pêche, épée, fourche, hache, pioche, pelle, laboureuse, marteau) | 8 poudres corrompue / item  |
| Mythique (canne à pêche, épée, fourche, hache, pioche, pelle, laboureuse, marteau)   | 16 poudres corrompue / item |

</details>

{% hint style="info" %}
À noter que les pioches à spawner et les bâtons de vente ne sont pas éligibles à la fonderie.
{% endhint %}
