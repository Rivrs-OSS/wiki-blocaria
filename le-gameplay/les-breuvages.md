# 🧪 Les breuvages

### Utilisation

Un breuvage vous permet d’obtenir **plusieurs avantages temporaires**, appliquant **différents effets** pendant une durée limitée.

Ces effets peuvent concerner divers aspects du jeu (bonus de progression, amélioration de rendement, avantages spécifiques, etc.), selon le breuvage utilisé.

Les breuvages doivent être **fabriqués** avant utilisation.\
Le détail de leur fabrication et de leur fonctionnement est présenté dans la section correspondante ci-jointe
