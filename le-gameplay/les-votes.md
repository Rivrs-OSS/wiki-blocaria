# 📨 Les Votes

Le système de vote vous permet de soutenir le serveur et de gagner des récompenses en votant sur différents sites. Chaque vote aide à améliorer la visibilité du serveur tout en vous donnant des avantages intéressants.

### Comment voter ?

1. <mark style="color:yellow;">**Accédez au Menu de vote**</mark> : Tapez la commande <mark style="color:yellow;">`/vote`</mark> pour ouvrir le menu de vote.
2. <mark style="color:yellow;">**Votez sur les Sites Partenaires**</mark> : Pour accéder aux sites de vote, rendez-vous sur le site de [**Blocaria**](https://blocaria.fr/vote) dans la catégorie [**Vote**](https://blocaria.fr/vote) <mark style="color:$danger;">liens site de vote</mark>. Vous trouverez les liens vers les différents sites partenaires permettant de voter pour notre serveur.
3. <mark style="color:yellow;">**Réclamez Vos Récompenses**</mark> : Revenez sur le serveur et utilisez la commande <mark style="color:yellow;">**`/vote claim`**</mark> puis accédez au menu des votes pour réclamer vos récompenses.
