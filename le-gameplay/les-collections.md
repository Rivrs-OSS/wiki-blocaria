# 📚 Les Collections

Le système de collection vous permet de répertorier tous les items d'une même catégorie. C'est collection consterne surtout les objets customs du serveur.\
Pour y accéder, vous avez le <kbd><mark style="color:yellow;">/collection<mark style="color:yellow;"></kbd>

Compéter une collection vous permettra d'obtenir un tag exclusif et 1 point pour le <kbd><mark style="color:yellow;">/classement<mark style="color:yellow;"></kbd>

<figure><img src="../.gitbook/assets/Capture d’écran 2026-02-15 à 08.18.28.png" alt=""><figcaption></figcaption></figure>

<mark style="color:yellow;">**Accès aux collections**</mark> : Utilisez la commande <mark style="color:yellow;">**`/collection <joueur>`**</mark> ou <mark style="color:yellow;">**`/profil`**</mark> <mark style="color:yellow;">**`<joueur>`**</mark> pour accéder à la collection d'un joueur spécifique. Vous pouvez facilement voir et comparer les collections des autres joueurs. Toutes les collections répertorient les objets personnalisés disponibles sur le serveur, vous encourageant à compléter votre collection autant que possible.

<mark style="color:yellow;">**Pourcentage de complétion**</mark> : Un pourcentage de complétion est affiché, indiquant à quel point ta collection est avancée. Cela te donne un objectif à atteindre et te permet de voir ta progression. Lorsque vous obtenez un nouvel objet, celui-ci est automatiquement débloqué dans sa collection.

{% hint style="info" %}
Ce système de collection vous encourage à explorer, échanger et compléter vos collections d'objets uniques. Ces collections permettent d'obtenir un tag qui apparaîtra dans le chat (avec le /tag).
{% endhint %}
