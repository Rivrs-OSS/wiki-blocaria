# ❓ Questions Fréquentes

### Pack de Textures

<mark style="color:yellow;">**"Mon pack de textures ne s'installe pas."**</mark>

1. Déconnectez-vous du serveur.
2. Retournez à la liste de vos serveurs multijoueurs.
3. Cliquez une fois sur _Blocaria_
4. Appuyez sur le bouton _Modifier_.
5. Vérifiez que la case _Pack de Ressources_ est bien activée.

### Boutique & Votes

<mark style="color:yellow;">**"Comment récupérer mes gemmes achetées sur le site ou mes récompenses de vote en jeu ?"**</mark>

* Utilisez la commande : **`/claim`**
* Vos gemmes ou récompenses de vote seront automatiquement ajoutées à votre compte.

### Discord

<mark style="color:yellow;">**"Comment obtenir le lien du Discord ?"**</mark>

* Utilisez la commande : <mark style="color:yellow;">**`/discord`**</mark>
* Un lien d’invitation s’affichera directement en jeu.

### Box

<mark style="color:yellow;">**"Comment inviter quelqu’un sur ma box ?"**</mark>

* Utilisez : <mark style="color:yellow;">**`/box invite (pseudo)`**</mark>

<mark style="color:yellow;">**"Comment quitter ma box pour rejoindre celle d’un ami ?"**</mark>

* Si d’autres joueurs sont présents : <mark style="color:yellow;">**`/box leave`**</mark>
* Si vous êtes seul : <mark style="color:yellow;">**`/box disband`**</mark> _(pensez à récupérer vos équipements avant !)_

### Hôtel des Ventes

<mark style="color:yellow;">**"Comment vendre un objet dans le /ah ?"**</mark>

* Tenez l’objet en main puis tapez : **`/ah sell prix`**

<mark style="color:yellow;">**"Comment trouver un objet précis ?"**</mark>

* Utilisez : **`/ah search name:<nom>`**

{% hint style="info" %}
_<mark style="color:$success;">**Astuce :**</mark>_ _<mark style="color:$success;">Pour estimer le prix moyen de revente d’un objet, utilisez :</mark>_ _<mark style="color:yellow;">**`/ah average`**</mark>_ _<mark style="color:$success;">avec l’objet en main.</mark>_
{% endhint %}

### Familiers

<mark style="color:yellow;">**"Comment améliorer mon Familier ?"**</mark>

* Activez votre familier via **`/pets`**, puis nourrissez-le avec des **bonbons** pour lui donner de l’expérience.

### Commerce

<mark style="color:yellow;">**"Comment vendre mes objets ?"**</mark>

* À l’hôtel des ventes via **`/ah sell prix`**
* Dans le **`/shop`** avec **`/quicksell`** .
