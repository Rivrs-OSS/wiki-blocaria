# 📍 Commandes

{% hint style="info" %}
<mark style="color:yellow;">`[...]`</mark> Signifie que le paramètre est obligatoire

<mark style="color:yellow;">`<...>`</mark> Signifie que vous pouvez mettre une chaine de caractères avec des espaces

<mark style="color:yellow;">`{...}`</mark> Signifie que le paramètre est optionnel
{% endhint %}

<details>

<summary><strong>Box</strong></summary>

<mark style="color:yellow;">`/box`</mark> : ouvre le menu de la box

<mark style="color:yellow;">`/missions`</mark> : ouvre le menu des quêtes communes aux membres pour agrandir la taille de la boxe avec plusieurs niveaux

<mark style="color:yellow;">`/box create [nom]`</mark> : crée une box

<mark style="color:yellow;">`/box disband`</mark> : supprime la box (disponible uniquement pour le chef, le disband est définitif le staff n'a pas la possibilité de le restaurer la box)

<mark style="color:yellow;">`/box leave`</mark> : quitte la box

<mark style="color:yellow;">`/box invite [pseudo]`</mark> : invite un joueur à rejoindre la box

<mark style="color:yellow;">`/box accept [pseudo/nom de l’île]`</mark> : accepte une invitation

<mark style="color:yellow;">`/box chest [1-9]`</mark> : ouvre les coffres de la box&#x20;

<mark style="color:yellow;">`/box bank`</mark> : ouvre l'interface de la banque

<mark style="color:yellow;">`/box bank logs`</mark> : affiche les logs de la banque

<mark style="color:yellow;">`/box deposit [somme]`</mark> : dépose de l’argent dans la banque

<mark style="color:yellow;">`/box withdraw [somme]`</mark> : retire de l’argent de la banque

<mark style="color:yellow;">`/box ban [pseudo]`</mark> : bannit un joueur de la box

<mark style="color:yellow;">`/box unban [pseudo]`</mark> : unban un joueur de la box

<mark style="color:yellow;">`/box expel [pseudo]`</mark> : expulse un joueur présent sur la box mais pas membre de la box

<mark style="color:yellow;">`/box kick [pseudo]`</mark> : kick un membre de la box

<mark style="color:yellow;">`/box border [couleur]`</mark> : change la couleur des bordures de la box (personnel)

<mark style="color:yellow;">`/box open`</mark> : ouvre la box au public

<mark style="color:yellow;">`/box close`</mark> : ferme la box au public

<mark style="color:yellow;">`/box coop [pseudo]`</mark> : ajoute un joueur en tant que coopérateur sur la box

<mark style="color:yellow;">`/box uncoop [pseudo]`</mark> : retire un coopérateur de la box

<mark style="color:yellow;">`/box promote [pseudo]`</mark> : promeut le rôle d’un membre de la box

<mark style="color:yellow;">`/box demote [pseudo]`</mark> : rétrograde le rôle d’un membre de la box

<mark style="color:yellow;">`/box setrole [pseudo] [rôle]`</mark> : change le rôle d’un membre de la box

<mark style="color:yellow;">`/box setleader [pseudo]`</mark> : transfère le lead de la box à un autre membre

<mark style="color:yellow;">`/box team {pseudo/nom de l’île}`</mark> : affiche des informations sur les membres d’une box

<mark style="color:yellow;">`/box info {pseudo/nom de l’île}`</mark> : affiche des informations sur une box

<mark style="color:yellow;">`/box name [nom]`</mark> : change le nom de la box

<mark style="color:yellow;">`/box upgrades`</mark> : affiche les améliorations de la box

<mark style="color:yellow;">`/box permissions`</mark> : modifie les permissions générales de la box

<mark style="color:yellow;">`/box permissions [pseudo]`</mark> : modifie les permissions d’un joueur sur la box

<mark style="color:yellow;">`/box settings`</mark> : ouvre le menu des paramètres de la box

<mark style="color:yellow;">`/box chat`</mark> : active le chat de la box

<mark style="color:yellow;">`/box c`</mark> : Toggle le chat de la box

<mark style="color:yellow;">`/box biome`</mark> : ouvre l'interface des biomes disponibles dans la box

</details>

<summary><strong>Jobs</strong></summary>

<summary><strong>Interactions entre Joueurs</strong></summary>

<mark style="color:yellow;">`/msg [pseudo] <message>`</mark> : envoie un message privé à un joueur

<mark style="color:yellow;">`/r <message>`</mark> : répond à votre dernier message privé

<mark style="color:yellow;">`/chattoggle`</mark> : active/désactive le chat global

<mark style="color:yellow;">`/ignore add [pseudo]`</mark> : ignore un joueur

<mark style="color:yellow;">`/ignore remove [pseudo]`</mark> : n'ignore plus un joueur

<mark style="color:yellow;">`/trade [pseudo]`</mark> : propose un échange avec un joueur

<mark style="color:yellow;">`/trade accept`</mark> : accepte une demande d'échange

<mark style="color:yellow;">`/trade deny`</mark> : refuse une demande d'échange

<mark style="color:yellow;">`/pay [pseudo] [montant]`</mark> : donne de l'argent à un joueur

<mark style="color:yellow;">`/pub <message>`</mark> : affiche un message de publicité dans le chat, disponible toutes les 20 minutes

<mark style="color:yellow;">`/pubtoggle`</mark> : active/désactive les publicités

</details>
